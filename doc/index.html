Getting Started

Overview

IBM Urbancode Deploy provides the ability for IT organizations to be able to continuously deliver their applications to production in a timely, secure, efficient manner. This integration extends that functionality and provides IT to target the Apprenda platform as an endpoint.

Release Notes

  • Version 1.0
    • List features here.

Features

  • Provides the ability to target Apprenda as an endpoint, enabling organiztions to deploy and update applications to the platform.
  • Deploy components to Bluemix, IBM's platform as a service (PaaS)
  • Create and update Bluemix services.
  • Bind Bluemix components to Apprenda applications, and vice versa.
